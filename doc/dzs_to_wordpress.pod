=head1  DESC

    将TXT导入wordpress空间

=head1 EXAMPLE

    dzs_to_wordpress.pl -W 顾漫 -b 何以笙箫默 -o hy.txt -c 言情 -w http://xxx.xxx.com  -u xxx -p xxx

=head1 OPTIONS

-W : writer name，作者名

-b : book name，书名

-o: 指定文本来源(可以是单个目录或文件)

-r: 指定分割章节的正则表达式(例如："第[ \\t\\d]+章")

-c : categories，小说类别，例如 原创

-t : tags，标签，例如 顾漫

-w : wordpress 地址

-u : wordpress 用户

-p : wordpress 密码

=cut
